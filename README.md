# react 后台管理项目前端
## 创建项目(使用create-react-app搭建)
- npm install -g create-react-app
- create-react-app react-admin
- cd react-admin
- npm start
- 在本地 http://localhost:3000 可以看到项目已经启动
## 构建目录结构
    src
        api        // ajax 请求相关目录
        components // 公共组件
        config    // 配置文件
        assets    // 静态资源
        pages     // 路由组件
        utils     // 工具函数目录
        index.js  // 入库文件
        App.js  // 根组件
